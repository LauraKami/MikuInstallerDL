## 🌟 What is MmdStart?

**MmdStart** is a "One-Click" installer powered by *Inno Setup*, designed specifically for **Windows 10 and 11 (64-bit)**.

No more manual searches for Visual C++ runtimes, DirectX 9, or missing `.dll` files: **MmdStart configures everything in a single step.**

## 🛠️ What Changed Since MikuInstaller?

* **New Installation Engine:** Rebuilt with *Inno Setup* for better system stability, compatibility, and easier updates.
* **100% Safe & Clean:** Verified flag-free by Microsoft Defender—no need to turn off your antivirus.
* **Re-run Support:** Easily launch the installer again anytime to toggle or add optional packages.
